# Web Search Skill

You can search the web to find current information. When the user asks about:
- Current events or recent news
- Real-time data (weather, stock prices, sports scores)
- Facts you're unsure about
- Anything that may have changed since your training

Use the `web_search` tool to find relevant information.

## Guidelines
- Always cite your sources when providing search results
- Prefer authoritative sources (official websites, news outlets, documentation)
- If search results conflict, mention the discrepancy
- Summarize findings concisely rather than dumping raw results

## Tools

### web_search
Search the web for current information.
Parameters: query (string) - The search query
